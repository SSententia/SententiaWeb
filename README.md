# Sententia's Website
Source code of my personal website <sup><sub>$\color{CadetBlue}{(duh)}$</sub></sup> with cool things to discover! The main branch is the source for GitHub pages, the alternative to my website. The source at Neocities has the "EXTERNAL" folder removed due to file type restrictions. The main branch will receive changes first; unstable or even broken changes may affect you if you’re using GitHub Pages. I recommend you use the main website.

Here's my [neocities website!](https://sententia.neocities.org/)

# Have an issue connecting to Neocities?
Neocities may be banned in your country. To get past their censorship freak policies, you may have to change your usual ISP DNS to known international DNS providers like Cloudflare (1.1.1.1) or Google (8.8.8.8). Works in Indonesia. I'm not sure if this works elsewhere
