    Windows 98b

    Windows 98b, the operating system nobody asked for! Created in 1998+27, it was a way for people to experience Windows 98 from a different perspective. It is up to date with the latest, revolutionary technologies. The great innovations we utilize include removing 98% of features the user won't probably use. This also includes leaving bugs unchecked as fixing them take up too much storage, time, and money! Who doesn't love pretty bugs anyway? The technologies we use allow us to reduce 19.98 megabytes of system data, which means roughly 420 more pictures can be stored. We hope you will have the best experience with this operating system.
